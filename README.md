# SOC1 - Security Operations Center (SOC) - TryHackMe

This repository contains documentation of my progress in the SOC1 course on TryHackMe. The course focuses on the basics of working in a SOC team, identifying threats, and analyzing and responding to security incidents.

## Objectives
- Learn the tools and techniques used in SOC operations.
- Develop skills in log analysis, network monitoring, and incident management.
- Practical solutions to challenges related to threat detection and response.

## Repository Structure
- **Task-X/**: Folders containing solutions for individual tasks (with `.md` files for descriptions, logs, and results).
- **writeup.md**: A summary of the entire course with my conclusions and achievements.

## Tools and Technologies
- **Wireshark**
- **Nmap**
- **Splunk**
- **Kibana**

## How to Use This Repository
1. Browse the individual task folders.
2. Review my solutions, results, and the techniques I applied.
3. For a full summary of the course, check the `writeup.md` file.
